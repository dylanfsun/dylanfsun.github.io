R Markdown

This is an R Markdown document. Markdown is a simple formatting syntax for authoring HTML, PDF, and MS Word documents. For more details on using R Markdown see http://rmarkdown.rstudio.com.

When you click the Knit button a document will be generated that includes both content as well as the output of any embedded R code chunks within the document. You can embed an R code chunk like this:

summary(cars)

notes on differences

It seems like the main difference between the notebook and markdown is that the notebook continuously generates .html updates after saving that can be viewed at any time, whereas the markdown must be compiled at the end with knit. Another difference maybe is that all plots and stuff will be automatically generated when you compile the markdown, whereas in the notebok you have to produce the plot inside the actual notebook for it to show up.
